# react-webpack5
react+webpack5搭建的项目

全局安装包
npm install

打包
npm run build

启动
npm start

端口号：8080


项目目录结构
  my-project
  |- config
    |- webpack.common.config.js
    |- webpack.dev.config.js
    |- webpack.prod.config.js
+ |- dist
+   |- js
+     |- bundle.js
  |- node_modules
  |- src
    |- app.js
  |- package.json
